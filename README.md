# AI-Assisted Smart Traffic Management System for Kochi

## Overview
This project is an AI-powered smart traffic management system designed for the city of Kochi. It leverages real-time computer vision and data analytics to optimize traffic flow and reduce congestion.

## Key Features
- **Real-time AI Vision:** Detected vehicles and traffic density using YOLOv8.
- **Network-level Traffic Orchestration:** Intelligent signal timing adjustments based on live traffic data.
- **Adaptive Signals:** Dynamic traffic light control to prioritize high-congestion lanes.
- **Emergency Green Corridor:** Automated prioritization for emergency vehicles.
- **OSM-based Digital Twin:** Visualizing traffic flow on a map of Kochi.

## Hackathon Demo Instructions
1.  **Backend:** Navigate to `backend/` and run `python main.py` (or equivalent).
2.  **Vision:** Navigate to `vision/` and run the vision processing script.
3.  **Frontend:** Open `dashboard.html` or start the frontend server.

## Installation
1.  Clone the repository.
2.  Install dependencies: `pip install -r requirements.txt`.
